# Chatbot Appointment System

This project is part of a thesis focused on designing and implementing
a chatbot-based system for booking and canceling appointments.

## Core Features
- Chatbot interface embedded in a website
- Appointment booking and cancellation
- Google Calendar integration
- Automation workflows using n8n
- Containerized deployment using Docker

## Technology Stack
- Node.js (Backend API)
- n8n (Workflow automation)
- Google Calendar API
- Docker & Docker Compose
- (Optional) React / WordPress frontend

## Architecture
The system is designed as a decoupled, containerized architecture
that allows easy integration into existing websites (e.g. WordPress).
